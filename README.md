# MedTime Frontend

This project is a frontend application built with Vue 3 for the MedTime / VetTime web app.

It implements a simple authentication flow and a basic home dashboard.

---

## Features

- Login with username and password
- Authentication handling
- Protected routes
- Home dashboard with sidebar layout
- User-friendly error messages
- Prevents double form submission

---

## Tech Stack

- Vue 3 (Composition API)
- Vue Router
- Tailwind CSS
- Vite
- Fetch API
